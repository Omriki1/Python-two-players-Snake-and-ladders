סולמות ונחשים הוראות הפעלה
יש להפעיל התוכנה ממחשב אחד או cmd וללחוץ על הפונקציה של ה host שם יפתח חלון נוסף שיש להזין את הIp ואת port אחר כך יש להריץ את הקוד ממחשב או חלון cmd נוסף שם יהיה צריך לבחור את הפונקציה של connect שם יש להכניס את אותו ip ו port שהזנתם קודם ואז המשחק יתחיל בחלון נפרד לשני השחקנים והשחקן שעשה host יהיה השחקן הראשון.
